# Agent Instructions

Read `../AGENTS.md` first when present. `AGENTS.md` files are authoritative for all agents; `CLAUDE.md` is only a Claude Code adapter.

## Git Discipline

Commit after every major change and push the scoped commit to `origin/development`.

Stage only files changed for the current task. Do not stage unrelated work or local assistant state.

Treat a major change as a coherent completed task or behavior change, not every tiny edit.
